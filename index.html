POSITION

Ideas principales

  • Los elementos posicionados estáticos no se ven afectados por las propiedades top, bottom, left, right y z-index.
  • Un elemento con position: static; no está posicionado de ninguna manera especial; siempre se posiciona de acuerdo con el flujo normal de la página.
  • position: relative; Un elemento con position: relative; está posicionado en relación con su posición normal.
  • El posicionamiento relativo desplaza un elemento respecto de su posición original establecida mediante el posicionamiento normal.
  • El desplazamiento de la caja se controla con las propiedades top, bottom, left, right y z-index.
  • Los demás elementos no ocuparán el espacio que deja el elemento desplazado. position; absolute; El posicionamiento absoluto se emplea para establecer de forma exacta la posición en la que se muestra en la caja de un elemento.

Apuntes

Propiedad Position. Especifica el tipo me método de posicionamiento utilizado para un elemento. position: static; Los elementos HTML se posicionan estáticos de forma predeterminada. Los elementos posicionados estáticos no se ven afectados por las propiedades top, bottom, left, right y z-index.

Cuando un elemento se posiciona de forma absoluta, el resto de elementos se posiciona de forma absoluta, el resto de elementos de la página se ven afectados y modifican su posición

Resumen

La propiedad position ayuda a modificar de distintas maneras la posicion de los elementos, moviendolos desde su lugar de origen o desde el mismo contenedor.

DISPLAY

  • Cada elemento de HTML tiene un valor de visualizacion predeterminada, para la mayoria de los elementos es block o inline.
  • block: Se extiende desde la izquierda hasta la derecha tanto como lo permita el documento.
  • inline: No comienza en una nueva linea y solo ocupa el ancho que sea necesario.
  • inline-block:Permite establecer el ancho y altura de un elemento.
  • none: Hace que el elemento no genere ninguna caja, por lo que desaparece del documento.
  • table: Permite asignar un comportamiento de tabla a los elementos que hay dentro.
  • table-row: Actua como un elemento fila.
  • table-cell: Estiqueta que trabaja como una celda de tabla.

CSS3

Ideas principales

  • Para algunas propiedades se deben usar los prefijos "-moz-", "-o-" y "-webkit-" para que funcionen en navegadores.
  • "@Font-hace" es una regla que nos permite cargar y usar cualquier fuente que necesitemos.
  • "Border-radius" es una propiedad que genera esquinas redondeadas para la caja de un elemento.
  • "Box-shadow" genera una sombra paralela o interna para un elemento, "Text-shadow" genera una sombra paralela para el texto de un elemento.
  • "Linear-gradient" es una propiedad para generar un degradado en el fondo.
  • "Radial-gradient" es una propiedad para generar un degradado en circulo o elipse desde un origen.
  • "Outline" es una propiedad que nos ayuda a generar un segundo borde alejado del borde original del elemento.

Apuntes

CSS3 es un intento de reducir el uso de codigo de "Javascript" y para estandarizar funcioes populares, CSS3 no solo cubre diseño y estilos web sino tambien forma y movimiento.

Gracias a estas propiedades agregadas podemos generar esquinas redondeadas a elementos, sombras paralelas e internas, transformaciones de elementos y reposicionamiento de distinos elementos.

Resumen

Las propiedades de CSS3 nos ayudan a utilizar funiones esteticas y funcionales sin la necesidad de utilizar javascript.

ANIMATION

Ideas principales

  • Tipos de atributos para la velocidad de la animacion; linear, ease, ease-in, ease-out, ease-in-out y cubic-bezier. Cada uno de estos tiene variaciones acerca la velocidad en como termina o inica la animacion.
  • animation-duration: Especifica el tiempo que dura una animacion.
  • animation-iteration-count: Especifica el numero de veces que se debe reproducir una animacion.
  • animation-delay: Especifica un retraso para el inicio de una animacion. Se expresa en segundos.
  • animation-play-state: Especifica si la animacion esta ejecutando.

Apuntes

Permite animar la mayoria de los elementos HTML sin usar JavaScript. Para aplicar animaciones dentro de CSS se debe añadir una regla @keyframes, una vez se vincule a un elemento se aplicara a su estilo.

En las animaciones podemos manipular distintos parametros como la duracion y el delay con el que comienza la animacion, las velocidades de como se movera, cambios de color, el numero de veces que la animacion se repetira, etc.

Resumen

Las animaciones de CSS nos ayudan a darle movimeinto, color y tiempo de duracion a distintos elementos dentro de nuestro documento HTML.

Referencias avanzadas CSS3

Ideas principales

  • La pseudoclase nth-child() nos permite encontrar un hijo en especifico.
  • Las pseudoclases child como; only-child, first-child y last-child ayudan a encontrar un hijo en una posicion especifica.
  • Los selectores de atributo como; (*=cadena), ($=final),(^=inicio). Segun el nombre de cada uno, podemos seleccionar un elemento sefun su valor especifico.
  • Los selectores de relacion: nos ayudan a identificar los hermanos adyecentes o generales de un elemento.

Apuntes

Las pseudoclases nos permiten manipular un elemento en especifico dentro de un contenedor, podemos utilizar distintos atributos para realizar cosas especificas, como buscar a el primer hijo en un contenedor, el ultimo dentro de una clase, cuando solo un elemento sea hijo unico, etc.

Tambien podemos realizar excepciones utilizando "not()" al señalar un elemento dentro de una clase, esto nos sirve por si queremos que todos los elementos de una clase tengan ciertas caracteristicaS excepto una en especifica.

Resumen

Estas referencias avanzadas nos ayudan a identificar y manipular a elementos muy especificos dentro de una clase.
